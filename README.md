# DSA2 Coursework Repository

This repository contains my solutions for the DSA2 coursework tasks:

- **Task 1:** N-Queens (Backtracking and Las Vegas) with board visualisation and optional fixed starting queen for backtracking
- **Task 2:** Minimum Spanning Tree (Kruskal) using NetworkX graphs with step-by-step console output and graph visualisations
- **Task 3:** Most Frequent Word (array-based and text-based versions with excluded words)
- **Task 4:** Recursive Palindrome check (case-insensitive)
